# Cut-cut(FrontEnd)
A RSS website able to cut and edit articles making collections.(Back End)

一个支持制作剪报的新闻聚合网站(后端部分)

## 技术栈
+ nodejs
+ express
+ mongodb
+ mongoose
+ cheerio
+ request

### 依赖
+ phantomjs

## How to run
```
$ git clone
```
```
$ npm install
download and install phantomjs
```
```
$ nodejs bin/www
```
